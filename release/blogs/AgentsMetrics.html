# Agents Metrics

The EBA team has released a feature to monitor the metrics of different agents across your system. These metrics allow you to inspect fine details of agent performance for different metrics at different points in time. These metrics are useful in the collaborative reasoning ecosystem which EBA operates in. It allows you to inspect the top callers for different parts of our execution pipeline to identify an anomalies among your agents. Unlike the execution workflow, these metrics target the overall historical performance of an agent, rather than its timeline for executing a particular question.

# Some use cases to consider

As an admin, if your agent is operating in a system where multiple production agents are enabled for a particular user, you will be to inspect within a historical sliding window, whether your agent performs as a bottle-neck.
